# Public Showcase

Positioning:

TriStack Alpha Agent demonstrates a credible, bounded trading-agent architecture instead of a black-box trading bot. It shows where the signal comes from, how the flash-loan math is computed, which risk gates block execution, and how every decision is recorded.

Judge-facing highlights:

- Strong USP: flash-loaned pool balancing between BNB Chain DEXs.
- Sponsor-native stack: CMC, Trust Wallet, BNBAgent SDK, BNB Chain.
- Safety-first design: no transaction by default.
- Explainability: decision hash and ledger record.
- Honest demo: real-mode blockers are documented.
